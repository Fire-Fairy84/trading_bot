# Proyecto Trading

Proyecto de aprendizaje para estudiar trading algorítmico paso a paso con Python y `backtesting.py`.

## Objetivo

Construir una base técnica simple y ordenada para:

- preparar el entorno local
- organizar el proyecto
- instalar dependencias mínimas
- validar ideas de swing trading mediante backtesting
